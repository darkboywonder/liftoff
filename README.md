# Animotion

This is an Animotion presentation.

## Setup
Install pnpm

```sh
npm install pnpm
```
or globally

```sh
npm install -g pnpm
```

Install dependencies.

```sh
pnpm i
```

Run the development server at http://localhost:5173/.

```sh
pnpm run dev
```

Build and preview deploy.

```sh
pnpm run build && pnpm run preview
```
